1.命名空间的引入

1.1 using static

在.NET6.0中，引入了一个新的语法，using static，假设我们在一个项目中要频繁进行数学运算，如下：

但这样用的多了就太过繁琐，此时我们可以使用using static System.Math;命名空间，相当于静态地使用了命名空间System.Math的这个类，然后就可以把Math.删掉,让代码显得更为简练。

但需要注意，这样的使用场景并不是很多，这种写法相当于把类中的所有静态方法全部暴露出来了，出现重名之类的问题就会变得比较繁琐。不过Math中的方法一般不会出现重名，但使用的时候仍需注意！

1.2 Global Usings

在C#10中，新增了一个语法global using，就是你可以在using前面加上个global，这样程序集中的多有类文件都可以识别到。
